Privacy Policy

Your Data Security is Our Top Priority

Last Updated: December 21, 2026

1. Information We Collect

When you use CodeSentinel AI, we collect information to provide you with the best analysis experience. This includes:

  • Authentication Data: Email address and profile information provided via Google or GitHub login.
  • Code Snippets: Code content you knowingly upload or paste for analysis.
  • Usage Data: Anonymous statistics about how you interact with our features (e.g., number of analyses, preferred languages).

2. How We Use Your Code

We understand that your code is your intellectual property. Our handling of your code is strict and transparent:

  • Ephemeral Analysis: Code snippets sent for analysis are processed in real-time by our Machine Learning models and the Google Gemini API.
  • No Long-Term Retraining: We do NOT use your private code snippets to train our public models or the Google Gemini model.
  • History Storage: Your analysis history is stored locally in your browser's LocalStorage for your convenience. Minimal metadata may be stored on our servers for dashboard statistics.

3. Third-Party Services

We utilize trusted third-party services to power our application:

  • Firebase (Google): For secure authentication and hosting.
  • Google Gemini API: For providing Generative AI code analysis.
  • GitHub API: For fetching repository details when you use the GitHub Analyzer.

Each of these providers has their own privacy policies, and data shared with them is governed by their respective terms.

4. Data Security

We implement industry-standard security measures to protect your data. All communication between your browser and our servers is encrypted via SSL/TLS. We do not store passwords or authentication tokens on our servers; these are handled securely by Firebase.
